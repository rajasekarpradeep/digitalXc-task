# digitalXc-task

Please clone the repository using 
```sh
git clone https://github.com/rajasekarpradeep/digitalXc-task.git
```

Install the dependencies
```sh
//frontend 
cd front-end
npm install

//backend
cd back-end
npm install

```

To run front end 
```sh
npm run dev
```

To run back end
```sh
node index.js
```

to test the logic in backend 
```sh
npm test
```
